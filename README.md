# 🔍 Bhavika's Digit Detective

> An interactive handwritten digit recognition application powered by an Artificial Neural Network (ANN), featuring a GUI that lets users draw digits and receive real-time predictions.

---

## 🚀 Overview

**Bhavika's Digit Detective** is a deep learning project that classifies handwritten digits (0–9) using an **Artificial Neural Network (ANN)** trained on the **MNIST dataset**.

The project includes a **Tkinter-based graphical interface** where users can draw digits on a canvas and instantly receive predictions from the trained model, demonstrating the integration of deep learning with an interactive desktop application.

---

## ✨ Features

- 🧠 ANN trained on the MNIST dataset
- ✍️ Interactive drawing canvas
- 🎯 Real-time handwritten digit prediction
- 🖥️ Simple and intuitive GUI
- 📊 High classification accuracy

## ▶️ Getting Started

Clone the repository and install required dependencies

```bash
git clone https://github.com/bhavika-sri02/Bhavikas-Digit-Detective.git
```

Run the GUI:

```bash
python digit_gui.py
```

Draw a digit, click **Predict**, and view the model's prediction instantly.
